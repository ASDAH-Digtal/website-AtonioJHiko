Chess is a board game involving two players. Each player controls either the white or black army. Each army consists of 16 pieces. The goal of each player is to checkmate their opponent's King, which ultimately wins them the game.

Players needed to play: 2
Amount of pieces each player can control : 16
Skills needed: Strategic thinking, tatics
